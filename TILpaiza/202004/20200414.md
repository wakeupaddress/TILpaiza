D118:こよみの変換

```
<?php
    // 自分の得意な言語で
    // Let's チャレンジ！！

    $one = fgets(STDIN);
    
    $two = fgets(STDIN);
    
    if($one="S"){
        echo 1925+$two;
    }
    elseif($one="H"){
        echo 1988+$two;
    }
?>
```
